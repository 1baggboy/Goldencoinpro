# GoldenCoin Limited v2.1.2

A secure, premium crypto platform for managing Bitcoin deposits, transactions, and KYC verification.

## Features
- Real-time BTC market data with high redundancy (Binance, CoinCap, BitPay).
- Gemini AI Market Desk for sophisticated insights.
- Personalized Daily Investment Strategies.
- Secure KYC and Transaction management.
- Admin dashboard for user oversight.

## Technical Stack
- React 19 + TypeScript
- Express Backend
- Firebase Authentication & Firestore
- Tailwind CSS 4.0
- Gemini AI Studio Integration
